<h1>MinhaLista</h1> 

Projeto de conclusão do Curso Técnico em Informática do Colégio Estadual Pedro Macedo.

Site para criar sua lista de anime e manga.
- Perfis
- Listas
- Ranking
- Favoritos
- Busca
- Baseado no MyAnimelist e Anilist
