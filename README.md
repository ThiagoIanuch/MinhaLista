<h1>MinhaLista</h1> 

Projeto realizado para a conclusão do Curso Técnico em Informática do Colégio Estadual Pedro Macedo.

MinhaLista é um site para criar sua lista de anime e manga.
- Perfis;
- Listas;
- Ranking;
- Favoritos;
- Busca;
- Baseado no MyAnimelist e Anilist.
